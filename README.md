# Bash-Terminal
A Linux/Bash terminal to run in a Linux terminal written in C.


 * DESCRIPTION: A shell to run 
 * just about everyting you
 * could in a normal terminal
 * - bash commands
 * - execute files
 * - change directory
 * 
 * 
 * COMPILE: gcc -o shell shell.c
 * RUN: ./shell
 * STOP: exit or ctr-c
